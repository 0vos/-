# Huffman Compression Software
> 2024 Winter 数据结构课程设计
```
5、哈夫曼压缩软件设计
【问题描述】
采用哈夫曼树求得的用于通信的二进制编码称为哈夫曼编码。利用哈夫曼编码对文本或图像进行数据压缩，设计数据压缩软件。
【设计要求】
设计基于哈夫曼编码的文本和图像压缩软件。
（1）采用静态链表的二叉树等数据结构。
（2）可以随机、文件及人工输入数据。
（3）创建哈夫曼树，生成哈夫曼编码和译码。
（4）源码、编码和压缩后的信息均以文件形式保存。
（5）可以查询和更新数据。
（6）其它完善性或扩展性功能。
```
该项目实现了一个轻量级的C++哈夫曼压缩软件，具体来说是后端的哈夫曼算法和压缩文件生成，前端显示用户界面进行输入和展示操作
